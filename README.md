# Sudoku-SAT-Solver

## Lancement du programme 

```./solve.sh grid1.txt```
 
 
## Variables propositionnelles

Nous avons des variables du type ```lx1cx2vx3``` avec ```x1``` le numéro de la ligne, ```x2``` le numéro de la colonne et ```x3``` la valeur. Donc par exemple ```l2c4v8``` représente la valeur 8 pour la 4ième case de la deuxième ligne.

## Formules propositionnelles 

Nous avons utilisé les formules propositionnelles proposé dans ce PDF : http://disi.unitn.it/~ldkr/ml2013fall/slides/2_propFormaliz.pdf
